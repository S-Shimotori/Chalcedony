#Chalcedony

## Description
Chalcedony records what time you "laboin" and "laborida" then tweets!

* laboin ...entering a laboratory
* laborida ...leaving a laboratory

Chalcedony tweets
* you've done laboin
* you've done laborida
* you feel like going home
